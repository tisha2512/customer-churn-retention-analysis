# Customer Churn & Retention Analytics

## Project Overview

This project analyses customer churn behaviour to identify why customers leave and how businesses can improve customer retention.

## Tools Used

- Excel
- SQL
- Power BI
- Python

## Dataset

The dataset contains customer information such as contract type, tenure, payment method, monthly charges, total charges and churn status.

- Created Churn Value
