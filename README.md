# Customer Churn & Retention Analytics

## Project Overview

This project analyses customer churn behaviour to identify why customers leave and how businesses can improve customer retention.

## Tools Used

- Excel
- SQL
- Power BI
- Python

## Dataset

The dataset contains customer information such as contract type, tenure, payment method, monthly charges, total charges and churn status.

## Day 1: Data Cleaning

Completed in Excel:

- Checked missing values
- Checked duplicate records
- Verified data quality
- Created Tenure Group
- Created Monthly Charge Group
- Created Churn Value

## Day 2 - SQL Analysis

The SQL phase focused on identifying customer churn patterns, high-risk customer segments and revenue impact.

### Key Findings

- Overall Churn Rate: 26.58%
- Revenue At Risk: $139.1K
- Highest Risk Contract: Month-to-Month (42.71%)
- Highest Risk Payment Method: Electronic Check (45.29%)

### High-Risk Customer Profile

The highest-risk customer segment was:

- Contract Type: Month-to-Month
- Payment Method: Electronic Check
- Churn Rate: 53.73%
